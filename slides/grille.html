Note formateur — Méthode professionnelle
La grille Bastien & Scapin est le standard académique francophone. Pour chaque action recommandée dans un rapport d'audit, on indique les numéros des critères violés (ex : "Critères 1, 2, 6, 8"). Cette notation systématique rend les audits reproductibles et défendables.

6.2 — Grille d'audit & priorisation

Échelle de sévérité des problèmes UX (référentiel professionnel) :

Niveau Description Action
0 Pas un problème UX Aucune action
1 Problème cosmétique Correction si le temps le permet
2 Problème mineur Priorité basse — prochain sprint
3 Problème majeur Priorité haute — sprint courant
4 Catastrophe Bloquer la mise en prod — correction immédiate
